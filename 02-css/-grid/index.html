Grid 网格布局

什么是 Grid？

Grid 是 CSS 最强的布局系统，可以同时控制行和列，实现二维布局。

Flexbox（一维）

主要控制一个方向（行或列）

123
Grid（二维）

同时控制行和列

.container {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr;  /* 3等分列 */
    grid-template-rows: 100px 100px;     /* 2行各100px */
    gap: 10px;                           /* 间距 */
}

1fr 2fr 1fr → 1 : 2 : 1 的比例

今日小结

  1. display: grid：开启网格布局
  2. grid-template-columns/rows：定义网格结构
  3. fr：比例单位，按份数分配空间
  4. repeat()：简化重复写法
  5. minmax()：响应式列宽，设置最小最大范围
  6. auto-fill：自动填充列数
  7. grid-column/row: span n：合并单元格
