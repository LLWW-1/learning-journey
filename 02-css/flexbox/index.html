Flexbox 弹性布局

什么是 Flexbox？

Flexbox 是一种一维布局模型，可以让元素在容器内灵活排列，解决传统布局（浮动、定位）的复杂问题。

核心思想： 让容器有能力改变子元素的宽度/高度/顺序，以最佳方式填充可用空间。

主轴与交叉轴

主轴 (Main Axis)
交叉轴 (Cross Axis)
  • 主轴：flex 项目排列的方向（默认水平从左到右）
  • 交叉轴：与主轴垂直的方向
