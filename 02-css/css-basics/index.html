CSS基础入门

什么是CSS？

CSS（层叠样式表）是用来给HTML"穿衣服"的语言。没有CSS的网页是黑白文字，有了CSS就可以控制颜色、大小、位置、动画等一切视觉效果。

核心思想： 结构与样式分离。HTML负责内容，CSS负责表现。

CSS的三种引入方式

  • 内联样式：写在标签的 style 属性里（不推荐，优先级最高但不好维护）
  • 内部样式表：写在 <style> 标签里（适合单页面）
  • 外部样式表：写在单独的 .css 文件里（推荐，可复用、易维护）

CSS选择器

选择器用来"选中"要美化的元素。

选择器 写法 作用
标签选择器p { }选中所有 <p>
类选择器.card { }选中所有 class="card"
ID选择器#special { }选中 id="special"（唯一）
后代选择器.card p { }选中 .card 里的所有 p

常用CSS属性

  • color：文字颜色
  • background-color：背景颜色
  • font-size：字体大小（px、rem等）
  • font-family：字体类型
  • text-align：文字对齐（left/center/right）
  • padding：内边距（内容到边框的距离）
  • margin：外边距（元素之间的距离）
  • border：边框
  • border-radius：圆角
  • box-shadow：阴影

盒子模型

在CSS中，每个元素都是一个"盒子"，由这几部分组成：

Margin (外边距)
Border (边框)
Padding (内边距)
Content (内容)
  • Content：内容区域（文字、图片等）
  • Padding：内边距，内容到边框的距离
  • Border：边框
  • Margin：外边距，元素之间的距离

记住： 一个元素实际占用的宽度 = width + padding + border + margin
