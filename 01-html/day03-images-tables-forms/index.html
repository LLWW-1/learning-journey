img标签属性：

  • src：图片地址（必须）
  • alt：图片加载失败时显示的替代文字（必须，对SEO和读屏软件友好）
  • width / height：设置宽高（单位像素）

表格标签说明：

  • <table>：表格容器
  • <thead>：表头区域
  • <tbody>：表格主体区域
  • <tr>：表格行（Table Row）
  • <th>：表头单元格（Table Header）
  • <td>：普通单元格（Table Data）

3. 表单标签 <form>

表单用于收集用户输入的数据，是前后端交互的基础。

表单标签说明：

  • <form>：表单容器
  • <input>：输入框，type属性决定类型（text/password/email/radio/checkbox等）
  • <label>：标签，for属性关联input的id
  • <select> + <option>：下拉选择框
  • <textarea>：多行文本输入框
  • <button>：按钮，type="submit"提交表单
